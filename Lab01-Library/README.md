## Lab01 — System Wypożyczalni Książek (Library)

### Wymagania
* Docker & Docker Compose

### Uruchomienie (z folderu projektu)
```bash
cd Lab01-Library
docker compose up --build
```

* Info: Przy pierwszym uruchomieniu baza danych zostanie automatycznie wypełniona przykładowymi danymi (Seed Data).

### Dostęp do aplikacji

Serwis	       Adres URL	                  Opis
Frontend	   http://localhost:8080	      Interfejs graficzny użytkownika (HTML/JS)
Backend API	   http://localhost:8000/docs	  Dokumentacja Swagger UI
