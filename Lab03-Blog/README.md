## Lab03 — Blog komentarze z moderacją (Blog)

### Wymagania
* Docker & Docker Compose

### Uruchomienie (z folderu projektu)
```bash
cd Lab03-Blog
docker compose up --build
```

* Info: Przy pierwszym uruchomieniu baza danych zostanie automatycznie wypełniona przykładowymi danymi (Seed Data).

### Dostęp do aplikacji

Serwis	        Adres URL	                  Opis
Frontend	    http://localhost:8082	      Interfejs graficzny użytkownika (HTML/JS)
Backend API	    http://localhost:8002/docs	  Dokumentacja Swagger UI
