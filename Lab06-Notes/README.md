## Lab06 — Notatki, tagi i wyszukiwanie (Notes)

### Wymagania
* Docker & Docker Compose

### Uruchomienie (z folderu projektu)
```bash
cd Lab06-Notes
docker compose up --build
```

* Info: Przy pierwszym uruchomieniu baza danych zostanie automatycznie wypełniona przykładowymi danymi (Seed Data).

### Dostęp do aplikacji
