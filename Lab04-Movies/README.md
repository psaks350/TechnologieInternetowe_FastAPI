## Lab04 — Filmy i oceny (Movies)

### Wymagania
* Docker & Docker Compose

### Uruchomienie (z folderu projektu)
```bash
cd Lab04-Movies
docker compose up --build
```

* Info: Przy pierwszym uruchomieniu baza danych zostanie automatycznie wypełniona przykładowymi danymi (Seed Data).

### Dostęp do aplikacji

| Serwis | Adres URL | Opis |
| :--- | :--- | :--- |
| **Frontend** | [http://localhost:8083](http://localhost:8083) | Interfejs graficzny użytkownika (HTML/JS) |
| **Backend API** | [http://localhost:8003/docs](http://localhost:8003/docs) | Dokumentacja Swagger UI |
