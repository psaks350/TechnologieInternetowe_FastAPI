## Lab05 — Kanban (kolumny i zadania)

### Wymagania
* Docker & Docker Compose

### Uruchomienie (z folderu projektu)
```bash
cd Lab05-Kanban
docker compose up --build
```

* Info: Przy pierwszym uruchomieniu baza danych zostanie automatycznie wypełniona przykładowymi danymi (Seed Data).

### Dostęp do aplikacji

| Serwis | Adres URL | Opis |
| :--- | :--- | :--- |
| **Frontend** | [http://localhost:8084](http://localhost:8084) | Interfejs graficzny użytkownika (HTML/JS) |
| **Backend API** | [http://localhost:8004/docs](http://localhost:8004/docs) | Dokumentacja Swagger UI |
